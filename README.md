# 📖 Quran API

A simple backend API for a Quran web application built with Node.js, TypeScript, and Hono.

---

## 🚀 Project Overview

This API provides Quran data and search functionality for a frontend application.

### Features:
- Get all Surahs and Ayahs
- Search Ayahs by English translation text
- Fast serverless API
- Built with TypeScript

---

## 🧱 Tech Stack

- Node.js
- TypeScript
- Hono
- Vercel Serverless Functions
- JSON Dataset
